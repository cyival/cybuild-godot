# Cybuild for Godot

A godot plugin for in-editor build support.

## Install

First, make sure you have installed cybuild and properly configured it in $PATH so that it could be accessed via cmdline.

Download the latest plugin in Releases and extract it into your project. It will be automatically enabled once if you started editing the project.

## Build

Requirements:

- Rust 1.90 or above
